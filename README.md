# 🏋️ Sports Planner

A personal weekly sports planner — built as a single HTML file, no dependencies, no server needed.

## Features

- **Routine templates** — define a weekly routine and assign it to a date range. When the range ends, create a new one with updated exercises
- **Weekly view** — navigate week by week, see which routine is active
- **Checklist per sport per day** — each day shows blocks per sport (Tennis, Gym, Cardio...) with exercises you can check off. Checks reset each week, exercises come from the template
- **Progress bar** per day based on completed exercises
- **Multi-level objectives** — macro goals, monthly objectives, and weekly micro-goals with checkboxes
- **Editable everything** — click any text to edit it directly
- **Export** — save as PNG image or print as PDF
- **Collapsible sidebar** with sport management

## Tech

Pure HTML + CSS + JavaScript. Zero frameworks, zero build steps.  
Uses [html2canvas](https://html2canvas.hertzen.com/) for PNG export (loaded via CDN).
